IMPORTANT: For maximum email client compatibility, SVG files should be converted to PNG format at both 1x and 2x resolutions. SVGs may not render correctly in Outlook and some mobile email clients.

CONVERSION NOTES
----------------
- All SVG files should be converted to PNG format for maximum email client compatibility
- Use 2x versions for @2x/@3x displays (retina)
- Consider creating fallback versions for older email clients
- Test all images in Litmus/Email on Acid before deployment

FILE NAMING CONVENTION
----------------------
For PNG exports, use the following naming:
- 1x: [filename].png
- 2x: [filename]@2x.png

Example:
- header-logo-gift.png (1x)
- header-logo-gift@2x.png (2x)
